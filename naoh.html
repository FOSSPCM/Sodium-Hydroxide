The Sodium-Hydroxide Tests.

This program is conducting a test of your browser's compliance with modern-day Web and Security standards. This is only a test.

These are only automated tests for seeing how well your browser supports the web standards and security practices for the modern web, each organized by category. You will see a "PASS," "FAIL," or "PARTIAL" next to each test name. You want to see "PASS" for every test. If you see anything other than just a "PASS," you can find more information by accessing your browser's console (usually this is done by right-clicking, or in the case of Mac users, "secondary-clicking," this page and choosing "Inspect," then go to the "Console" tab in the box that shows up).

Please understand that there is only so much that an automated test can do. Manual testing is needed for a more thorough test. Nevertheless, this should be good enough to convey what your browser supports in the grand scheme of things.
